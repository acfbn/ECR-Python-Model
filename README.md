# Ecuación de Compatibilidad Relacional (ECR) v1.0 🚀

Este proyecto es un modelo matemático desarrollado en **Python** que analiza la viabilidad de una relación humana utilizando principios de ingeniería, como el balance de cargas y multiplicadores de eficiencia.

## 🧠 El Concepto
Inspirado en modelos como el **Número de Reynolds**, la ECR busca cuantificar sistemas complejos (relaciones) mediante variables estructurales y moduladores de riesgo.

### La Fórmula Maestra
$$NER = \left[ \left( \frac{M + B + 0.5A + S + 0.7E}{4.2} \right) \cdot \sqrt{\frac{C}{10}} \right] - 0.01I^3$$

## 🛠️ Características
- **Análisis Estructural:** Evalúa Madurez, Compatibilidad, Antecedentes, Estabilidad y Emoción.
- **Factor de Eficiencia:** Aplica una raíz cuadrada a la reciprocidad; si no hay esfuerzo mutuo, el sistema colapsa.
- **Colapso Cúbico:** Penaliza la impulsividad y toxicidad de forma exponencial.
- **Condición de Frontera:** Detecta cizallamiento vectorial (proyectos de vida opuestos).

## 🚀 Cómo ejecutarlo
1. Asegúrate de tener Python 3.x instalado.
2. Clona este repositorio.
3. Ejecuta `python ecr_main.py` en tu terminal.
